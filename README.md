<h1 align="center">Generion mobile client</h1>

Install node (https://nodejs.org/en/download/)
Install cocoapods (https://cocoapods.org/)

In console in project dir run:
- npm install
- cd ios
- pod install
- cd ..
- react-native link
- react-native run-ios
